# BackTop 回到顶部

## API

| 属性     | 说明                                               |  是否必须  | 类型     | 默认值  |
| -------- | ----------------------------------------------------- | -------- | -------- | ------- |
| target     | 设置需要监听其滚动事件的元素，值为一个返回对应 DOM 元素的函数                                |  否    | function  | -       |
| visibilityHeight | 滚动高度达到此参数才出现BackTop按钮                                        | 否 | number  | 200   |
| onClick  | click 事件的 handler                                 | 否 | function | -       |
